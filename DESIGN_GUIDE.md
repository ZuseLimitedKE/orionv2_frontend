
## Animation Guidelines

### Transitions
- Duration: 150-300ms
- Easing: ease-in-out
- Properties: colors, transforms, opacity

### Loading Spinners
- Size: 16px (buttons), 24px (cards)
- Color: Current text color
- Animation: Continuous spin

### Skeleton Loaders
- Background: Muted gray
- Animation: Shimmer effect
- Duration: 1.5s loop
